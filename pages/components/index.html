  • 二次封装 element-plus 的 Table ，完全兼容 el-table (opens new window) 所有 api 并提供灵活的配置项以及完善的类型提示，完全兼容 nuxt3 (opens new window)，完全支持 jsx 语法，无需将代码都写在 template 里

查看示例 (opens new window) 查看源码 (opens new window)

# pure-admin-descriptions

  • 二次封装 element-plus 的 descriptions (opens new window) 组件，提供灵活的配置项，完全支持 jsx 语法，无需将代码都写在 template 里

查看示例 (opens new window) 查看源码 (opens new window)

  • 基于 element-plus 中 el-dialog (opens new window) 组件开发的函数式弹框，无需在 template 里写弹框，使用更便捷
